# John Doe - Resume Website

A simple HTML resume website for John Doe, featuring professional experience, skills, projects, and contact information.

## Setup Steps

1. **Clone or Download the Project**
   - Download the project files to your local machine
   - Navigate to the `assignment1` folder

2. **No Installation Required**
   - This is a static HTML file with no dependencies
   - No npm packages or external tools needed

3. **Open in Browser**
   - Locate the `index.html` file
   - Double-click the file to open it in your default browser
   - Or right-click and select "Open with" to choose a specific browser

## Usage

### Viewing the Resume
1. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge, etc.)
2. The resume will display with the following sections:
   - **Header**: Name and professional title
   - **About Me**: Professional summary
   - **Skills**: List of technical skills
   - **Experience**: Work history in table format
   - **Projects**: Detailed project descriptions
   - **Education**: Educational background
   - **Contact**: Email, phone, and location information

### Customizing the Content
1. Open `index.html` in a text editor (Notepad, VS Code, etc.)
2. Edit the content sections:
   - Replace "John Doe" with your name
   - Update all sections with your own information
   - Modify dates, companies, and projects as needed
3. Save the file
4. Refresh the browser to see your changes

### Adding Styling (Optional)
- Currently, the file contains only HTML with basic table borders
- To add CSS styling, you can:
  - Create a separate `style.css` file
  - Add a `<link>` tag in the `<head>` section
  - Or add inline `<style>` tags to the HTML file
